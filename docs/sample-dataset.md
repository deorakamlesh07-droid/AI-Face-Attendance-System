# Sample Dataset Guidance

Use 8-15 well-lit face images per student.

- Capture front-facing images plus slight left/right angles.
- Avoid sunglasses, heavy blur, and very low light.
- Keep one face per image for training.
- Use JPEG or PNG images at 640x480 or higher.
- Retrain when appearance changes significantly.

Suggested naming:

- `rollno_01.jpg`
- `rollno_02.jpg`
- `rollno_03.jpg`

Recommended test flow:

1. Seed the database and log in as admin.
2. Create or confirm student records.
3. Upload 8-15 training images for a student from the admin or teacher panel.
4. Start a teacher attendance session.
5. Present one or more faces to the webcam and run frame scans.
6. Stop the session and review exports, logs, and alerts.
